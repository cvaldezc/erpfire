Devoluciones


Con Guia de Remision
PROCESAR DEVOLUCION
Detalle

Con esta opción se realiza la devolución a almacen con el número de guia referencial de donde se devuelve los accesorios de dicha guia. Adicionalmente para que se realice la devolución se requiere del ingreso de un número de guia para el registro de devolución.


Sin Guia de Remision
PROCESAR DEVOLUCION
Detalle

Con esta opción se realiza una nota de devolución al no tener el número de guia referencial se ingresa los accesorios manualmente. Adicionalmente en un segundo paso se puede seleccionar los materiales que se pueden reciclar para eso se registra una guia de remisión para el ingreso al inventario.
